## 🚀 INSTALACIÓN RÁPIDA (10 minutos)

### PASO 1: Ejecutar Script (5 min)

1. **Transferir todos los archivos** a tu PC Windows
2. **Abrir PowerShell como Administrador:**
   - Click derecho en botón Windows
   - "Windows PowerShell (Administrador)"
3. **Ejecutar:**
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force
cd C:\
.\CS2_OPTIMIZACION_NUCLEAR.ps1
```
4. **Escribir "SI"** para confirmar
5. Esperar 3-5 minutos
6. **REINICIAR PC**

### PASO 2: Configurar NVIDIA (2 min)

**Click derecho en Escritorio → Panel de Control NVIDIA**

**Manage 3D Settings → Program Settings → Counter-Strike 2:**
```

Low Latency Mode               → Ultra
Vertical Sync                  → Off
Texture Filtering - Quality    → Performance
```

Steam → CS2 → Properties → Launch Options:

## 🆘 SOPORTE Y TROUBLESHOOTING

### Problemas comunes:

**"No puedo ejecutar el script"**
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force
```

**"FPS sigue bajo después de optimizar"**
1. Verificar DLSS activado (Quality Mode)
2. Cable conectado a GPU (no motherboard)
3. NVIDIA Panel: Power Management → Max Performance
4. CS2: Multicore Rendering → Enabled

**"Microstutters / Frame drops"**
1. Cambiar `fps_max 0` → `fps_max 300` en autoexec
2. Cerrar Discord overlay
3. Cerrar software RGB
4. Limpiar shader cache

**"Latencia alta"**
1. Usar cable Ethernet (no WiFi)
2. Verificar: `net_graph 1` en CS2
3. Reset network stack (ver guía)

### Revertir cambios:

**Punto de restauración:**
```
Windows + R → rstrui.exe
Seleccionar: "PRE-CS2-NUCLEAR-OPT"
```

**Reactivar servicios:**
```powershell
# Windows Update
Set-Service wuauserv -StartupType Automatic
Start-Service wuauserv

# Windows Search
Set-Service WSearch -StartupType Automatic
Start-Service WSearch
```

## 📈 BENCHMARKING

### Antes de optimizar:

1. **Anotar FPS promedio** (CS2 Deathmatch 10 min)
2. **Test DPC Latency** (LatencyMon 5 min)
3. **Ping base** (`ping 8.8.8.8 -n 100`)
4. **Captura GPU-Z** (GPU load, temp)

### Después de optimizar:

1. **Mismo test FPS** (comparar)
2. **Test DPC Latency** (debe ser <100μs)
3. **Ping test** (debe ser similar o mejor)
4. **GPU-Z** (verificar boost clocks mantenidos)

### Herramientas recomendadas:
- **LatencyMon** - DPC latency
- **MSI Afterburner** - FPS + monitoring
- **HWiNFO64** - Temperaturas
- **GPU-Z** - Info GPU
- **CS2 net_graph 1** - FPS + ping in-game
